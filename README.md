# Heart-disease-prediction-ml
Hello everyone this project is Heart disease risk prediction using machine learning this project was build by Nishanth G(25cs169),Nishanth M(25cs170),Poobalan K(25cs183),Nishandhi M(25cs168),Prakalya E(25cs188),Nidrashri V(25cs163),Monisha sri E(25cs150)

 PROBLEM STATEMENT: It is a system that takes a person's health details using machine learning .It predicts heart disease risk (low/medium/high).The dataset has 1198 patients .Each patients has 11 inputs like age,BP,cholestrol,etc...If the result is 0,the patient has no disease.If the result is 1, the patient has disease.
 The important tools we used are pandas,numpy,scikit-learn,flask,pickle.pandas reads the data,numpy handles numbers,scikit-learn does ML,Flask makes website,pickle saves the model.
In simple ,our project works on user data to ML model to probability to risk level to result on website.
